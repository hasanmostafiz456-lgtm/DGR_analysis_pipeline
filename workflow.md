# Workflow

## Main steps
1. RT anchoring
2. Mapping RT candidates to genomic coordinates
3. Extraction of RT-centered windows
4. TR/VR repeat calling
5. A-to-N signature filtering
6. Selection of primary TR/VR loci
7. Remote target search
8. Protein-level mapping and alignment
9. Pfam-based context interpretation

## Main tools
BLAST, Prodigal, MAFFT, HMMER, and Pfam.

## Notes
This repository focuses on custom analysis scripts, summary generation, and selected example outputs from the DGR workflow.
